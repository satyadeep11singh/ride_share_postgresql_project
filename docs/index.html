üìÅ Project Structure

This analytics dashboard is built on a PostgreSQL data warehouse using:

  • Star Schema: 3 dimension tables + 1 fact table (normalized design)
  • ETL Pipeline: 7 SQL scripts for data transformation and enrichment
  • Window Functions: LAG() for sequential driver analysis
  • Python Visualization: Plotly for interactive HTML dashboards
  • Data Privacy: PII masking, email/phone hashing, name anonymization
